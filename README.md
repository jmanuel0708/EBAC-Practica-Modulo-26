# Proyecto Módulo 26 - EBAC

Calculador de edad basado en TypeScript. Este proyecto toma una fecha de nacimiento y calcula la edad del usuario.

## Descripción

Este es un proyecto de práctica del Módulo 26 sobre TypeScript de EBAC. La aplicación calcula la edad de una persona a partir de su fecha de nacimiento, teniendo en cuenta el mes y el día actual.

## Requisitos Previos

Antes de usar este proyecto, necesitas tener instalado **Node.js**. Si aún no lo tienes, sigue estos pasos:

### Instalar Node.js

1. Ve a la página oficial: [https://nodejs.org/](https://nodejs.org/)
2. Descarga la versión **LTS** (recomendada para la mayoría de usuarios)
3. Ejecuta el instalador y sigue las instrucciones en pantalla
4. Para verificar que se instaló correctamente, abre una terminal y ejecuta:
   ```bash
   node --version
   npm --version
   ```

## Instalación

Una vez tengas Node.js instalado, sigue estos pasos para instalar el proyecto:

1. **Clona o descarga el repositorio**
   ```bash
   git clone https://github.com/jmanuel0708/EBAC-Practica-Modulo-26
   cd Proyecto
   ```

2. **Instala las dependencias**
   ```bash
   npm install
   ```

   Esto instalará las dependencias necesarias (en este caso, TypeScript).

## Cómo Usar

### Ejecutar el programa

Para ejecutar la aplicación, usa el comando:

```bash
npm start
```

Este comando ejecutará el programa que calcula la edad para las fechas de nacimiento de ejemplo incluidas en el código.

### Salida esperada

Cuando ejecutes el programa, verás algo como:

```
El usuario con fecha de nacimiento el 08 de Julio del 2002 tiene 21 años
El usuario con fecha de nacimiento el 03 de Diciembre del 2007 tiene 18 años
El usuario con fecha de nacimiento el 30 de Marzo del 2000 tiene 26 años
```

## Tecnologías Utilizadas

- **TypeScript** - Lenguaje de programación tipado basado en JavaScript
- **Node.js** - Entorno de ejecución de JavaScript
